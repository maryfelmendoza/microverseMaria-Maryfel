Introduction

Charging a tokenized card

Everyone likes to receive payments, this is why we start with a tutorial to see how to do it. This tutorial uses the Openpay.js library to send the card information directly to Openpay. Using this library, besides being the easiest way to save a card, it minimizes the scope of a PCI Compliance certification.

Steps

  • Antifraud system implementation to generate "device_session_id"

  • Create a token of debit or credit card using Openpay.js

  • Send charge information to your server

  • Send a charge request to Openpay

  • Openpay will sent a charge request to issuing bank and will answer
